Setting Up a Feeding Location

1. Move the robot to the starting position of the feeding area
2. Click "Get Starting Position" to record it
3. Move the robot to the end of the feeding area
4. Click "Get Ending Position" to complete the path

Setting Up a Waiting Position

1. Move the robot to the desired waiting position
2. Click "Get Position" to record it
3. This is where the robot will wait when not actively feeding

Setting Up a Loading Position

1. Move the robot to the starting position near the dispensers
2. Click "Get Starting Position" to record it
3. Move the robot to the end position
4. Click "Get Ending Position" to complete the loading path
